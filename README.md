# slack-fathom-crono

Slack bot for automated meeting follow-ups with Fathom, Gmail, Calendar, and Crono CRM integration

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Run tests
npm test
```
